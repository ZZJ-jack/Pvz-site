常见问题

关于 PVZ Python 中文复刻版的一些常见问题解答

PVZ Python 中文复刻版保留了原版的核心玩法和内容，并在持续更新中，但使用Python语言重新开发，适配现代操作系统，并提供完整的中文支持。
是的，PVZ Python 中文复刻版是完全免费的开源项目，不包含任何广告或内购。你可以自由下载、使用和分享游戏，也可以参与到项目的开发中。
我们提供两种版本供您选择：

安装版 (Setup)：下载后双击运行安装程序，按照提示完成安装即可。安装版会自动创建桌面快捷方式。

便携版 (Portable)：下载后即可运行，无需安装。便携版适合需要随身携带游戏的用户，可以直接放在U盘中使用。
当然欢迎！项目托管在Gitcode（Gitee和Github有镜像）上，你可以提交Issue报告bug或建议新功能，也可以Fork仓库并提交Pull Request贡献代码。关于素材贡献，请查看项目贡献指南中的具体要求。
目前没有开发移动设备版本的计划，PVZ Python 中文复刻版仅针对Windows平台开发和优化。
